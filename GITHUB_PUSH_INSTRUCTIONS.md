  - Copy the token and paste it as password

## 🆘 Troubleshooting

**"Permission denied (publickey)"**

- You need to set up SSH keys or use HTTPS with token
- Use HTTPS URL instead: `https://github.com/...`

**"Authentication failed"**

- Don't use GitHub password, use Personal Access Token
- Get token: https://github.com/settings/tokens

**"Repository not found"**

- Check the URL is correct
- Make sure repository exists on GitHub
- Check you're logged in as correct user

**"Updates were rejected"**

- Someone else pushed changes first
- Run: `git pull origin main`
- Then: `git push origin main`
