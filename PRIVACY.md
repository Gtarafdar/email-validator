# Privacy Policy

**Last Updated:** March 12, 2026

## Our Commitment

This Email Validator tool is designed with **privacy as the top priority**. We believe email validation shouldn't require compromising your data.

## What Data We Collect: ZERO

- ❌ **No emails stored** - Validated and immediately discarded
- ❌ **No validation results saved** - Returned to you only
- ❌ **No user tracking** - No analytics or cookies
- ❌ **No logs** - SMTP verification not logged
- ❌ **No third-party services** - Direct verification only

## How It Works

1. Email received via API → Processed in RAM
2. SMTP verification performed
3. Result returned to you
4. **All data deleted from memory**
5. Nothing written to disk

## Server Details

- **Hosting:** Self-hosted on Contabo VPS (Germany)
- **Data Retention:** Zero
- **Encryption:** HTTPS supported (optional nginx)
- **Access Control:** API key required

## GDPR Compliance

- **Right to Access:** No data exists to access
- **Right to Deletion:** Nothing to delete
- **Right to Portability:** No stored data
